## 使用內容

### 第一章 登入系統  

登入系統，主要是呼叫 DoLogin 這個函式。這個  
函式包含了三個參數：帳號(user)、密碼(password)、專案(project)  

```
_User.Login("test", "test", "justtest");
```

其中的專案代碼，需由系統架設人員提供，才能正確  
登入。登入之後，該登入的有效時限為 24小時，超過  
之後，需要重新進行登入，才能持續運作。
